## 💫 About Me

嗨！我是一名專注於 **嵌入式系統（Embedded Systems）** 與 **機器學習/時序資料分析（Machine Learning / Time-Series Data）** 的開發者。熱衷於將硬體感測器訊號與深度學習模型結合，實現各種智慧物聯網與穿戴式裝置的應用。

- 🔭 **近期研究**：基於 IMU 感測器與深度學習（GRU / Autoencoder）的穿戴式疲勞辨識系統。
- 🛠️ **核心專長**：微處理器核心開發、時序訊號特徵萃取、Linux 開發環境建立與伺服器維護。
- ⛰️ **休閒愛好**：熱愛登山與戶外活動，也是一名業餘無線電愛好者。
